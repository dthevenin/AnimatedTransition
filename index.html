Animated Transition

Introduction

Le but du système de transition animée est de:

  1. Proposer un système permettant d'animer des transitions en IHM
  2. Faire une API de programmation orientée programmation d'IHM
  3. Etre compatible avec le dataflow de VS Toolkit
  4. Etre compatible avec le PAR et SEQ de VS Toolkit
  5. Etre performant sur mobile


Choix

  1. Séparer et formaliser le timming, accéléaration et fonction de transformation
  2. ...

Helpers

animateTransition (temp API)

Summary

Syntax

animateTransition = function (obj, property, options);

Example

Rotate an object 'obj' from 0deg to 200deg, 2 times with 10 steps, during 5s (total 10s) 
var myAnim = animateTransition (obj, 'rotation', {
  duration: 5000,
  pace: Pace.getLinearPace (),
  steps: 10,
  repeatDur: 2,
  trajectory: new Vector1D ({values: [0, 200]}).init ()
});

Use Task_PAR and Task_SEQ

Syntax


var anim2 = animateTransition (...), ...;

// anim3 will start when anim1 and anim2 will finish
// anim1 and anim2 will start at same time

var seq = vs.seq (vs.par (anim1, anim2), anim3);
seq.start ();
